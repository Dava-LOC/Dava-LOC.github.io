Version History

PostedOctober 8, 2021
UpdatedOctober 13, 2021
Bydava-LOC

September 2021 Release v4/350.13

Release Improvements  

  • All Option and plugin EXE and DLL files will now display the latest version, to eliminate  confusion that you may be running mixed versions. 
  • eSuite specific training video updates to be released to the Community page  
  • SCO Parsing Doc available on the Community page  

eSuite Item 

  • New grid to view item price, cost and margin history 
  • Enhancements to grid columns and wizard criteria  

eSuite System  

Xchfile configuration can now be configured in setup console  

SMS Payment  

Ability to move your store points and accounts receivable balance to a more efficient centralized location  

eStore/mStore  

  • General enhancements / bug improvements See your LOC representative for a Demo on eCommerce  

eWIC 

  • Now supporting Priced to Clear Concept  

PinPad 

  • MX/M400 with 3DES certification complete- ready for pilot  
  • XPI5400 complete for Worldpay ISO

mFloor  

  • SETUP.INI: [Device_Scan_MFLOOR] section has been added to parse UPC 
  • Base Right Profile: Limit the user from seeing and editing prices 

Customer Deploy Check  

  • Improvement to ensure that the customers are in sync no matter if the  profile modifications are coming from Host/ Stores/Lanes or eStore. This also respects GDPR if using this configuration 

SCO/Connected Payments  

  • EBT is available now when using CP  

NCR/Fastlane 

  • New Feature- Use Quick Pick to manually capture customer ID  
  • Now able to run APPcard/Prologic on Fastlane 

Toshiba/ CHEC 

  • Ability to use DL for Age Verification  

Scalelink 

  • New Feature: Use Scale PLU as the reference for ingredients and  nutrifacts rather than forcing UPC. 

Canadian Bank

  • Moneris semi-integrated using Veriifone P400 Pinpad now available

June 2021 Release v450.12

Updated modules

eSuite UX

  • Now supports single sign-on to prevent having to login every time you switch eSuite flavor
  • Visual confirmation when deploying items

eSuite Wizard

  • Added search criteria available
    • Multi words search
    • UPC partial string

eSuite Grid

  • New ability to combine multiple columns when sorting grid content
  • Additional fields available to grid result

eSuite Batch

  • Batch selection is now done from a grid rather than popup screen with much more information about batch
  • Batch Number now offers more detailed information
  • Added Grab fields function to Price, Cost, POS and Main Batch Wizards
  • Added Copy Paste function to the POS and Main Wizards

eSuite Core Item

  • When assigning a PromoID to a list of Items, you can now see all promoID’s before creating a new one
  • Promo ID will now prompt a search pull down window when assigning & unassigning Promo ID’s

eSuite Buy

  • Initialize Inventory now available
  • Added Charge to Department

mFloor

  • Every tile can now be managed with base rights per operator level
  • The Zebra TC21/TC52 are now supported when using Enterprise Browser
  • Inventory: Can now edit ‘comment’ field of the transaction when creating an inventory document
  • Picking: Displays more information about the transaction when selecting Picking transaction
  • Picking: Single and multi picking are now combined in the same functionality

xStore/Kiosk

  • New button editor providing ability to customize home screens (tiles)
  • Addition order ability (modifier): slice qty for deli/cheese and per unit selection for scalable items

eSuite Account Receivable

  • Now able to centralize charge accounts

eSuite System

  • Added interactive questions during options/plugins installation
  • Dialogs are now displayed as a pop-up in order to configure the required switches when installing an option using samples driver
  • Now configurable – Link_AWG; Link_SuperValu_SIL; Link_SuperValu_DX; Link_Krasdale; Link_LivingNaturally; Link_White_Rose

eSuite Manage

  • Gift Card – Balance grid is now available with full address

eSuite Care

  • Customer “Display Changes” now available. With check box to select which changes to Deploy

January 2021 Release v450.11

Updated modules

eSuite Deploy

  • New deployment wizard Ability to select which items to deploy

eSuite Customer

  • Alternative customer table is now relying on Popup.ini to help fill record

eSuite Report

  • Ability to add Like Code column in Sencha grid

eSuite User Rights

  • Manage: Now includes 180+ entries user levels 1-7
  • Care: Now includes 160+ entries user levels 2-7
  • System: Now includes 20+ entries user levels 1,5 & 7

xStore

  • Email sent to consumers can now include from alias information based on store detail table

Kiosk

  • Advanced modifier: allows automatic question advancement at Kiosk

ApiPerWinRaw

  • Receipt printing now supports Esc/pos command for visual formatting when running SmsHttp service Supports barcode and logo printing

2020 Release v450.04

Updated modules

SMSPayment

  • Now multi-threaded to serve more clients at the same time and to be compatible with the new ApiGenericIP multithread and SMSHttps.

Back Office

  • LOC SMS mFloor to replace the PDA and RF options
  • New Deploy SMSRSP option for store to host item deployment

eSuite – Item Grid

  • Using the ReadOnly.Ini for record ownership eSuite and Server

eSuite – Help

  • eSuite now contains online help 4.5.0.4

eCommerce

  • New Option_eCommerce for eCommerce emails and document management
  • New Deploy xStore for Host to eCommerce item management
  • Ability to have electronic coupons as Featured items 4.5.0.4
  • Improve the coupon details to display the e-coupon criteria 4.5.0.4
  • Allow buying products directly from the e-coupon criteria 4.5.0.4
  • Allow displaying items from a department, subdepartment or PID inside an e-coupon criteria 4.5.0.4

eSuite System

  • SMSNet service – the all-new LOC SMS Load Balancer for eCommerce servers
  • New ApiXchFTP DLL to replace the executable from LOC SMS 3.5.0
  • New ApiXchScale DLL to replace the executable from LOC SMS 3.5.0
  • SMSHttps – the all-new LOC SMS core for version 4.5.0:
    • Enhanced documentation about replacing the Launchpad application by the SMSHttps service 4.5.0.4
    • New parameter to keep SQL connection alive in a cloud environment such as Azure 4.5.0.4

      SQLHeartBeat=18
    • Following latest security best practices (reference https://robotattack.org) 4.5.0.4
    • Added totalizer 2, 3, 3122, 3123, 3132, 3133 and 8101 to TRS_ tables for loyalty analysis 4.5.0.4
